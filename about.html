Mathematical Concepts

Measures of Central Tendency (Ungrouped Data)

Measures of central tendency describe a central value for a set of data. The three main measures are:

  • Mean: The average of a dataset, calculated by dividing the sum of all values by the number of values.
  • Median: The middle value when the data is ordered from smallest to largest.
  • Mode: The value(s) that appear most frequently in the dataset.

Example:

Consider the dataset: 4, 8, 6, 5, 3.

  • Mean: (4 + 8 + 6 + 5 + 3) / 5 = 26 / 5 = 5.2.
  • Median: 5 (middle value when the dataset is ordered: 3, 4, 5, 6, 8).
  • Mode: No mode (all values appear only once).

Fundamental Counting Principle

The Fundamental Counting Principle states that if one event can occur in \(m\) ways and a second event can occur independently in \(n\) ways, then the total number of ways the two events can occur is \(m \times n\).

This principle is used to calculate the total number of outcomes in a sequence of events.

Example:

If you have 3 shirts (red, blue, green) and 2 pants (black, white), the total number of outfits is:

  • Red shirt with black pants
  • Red shirt with white pants

  • Green shirt with black pants
  • Green shirt with white pants
